This library project has three external library dependencies:

1. Add the 'apache-mime4j-core-0.7.jar' to your project build path from Core/libs
folder. See here for more information on how to add jars to a project 
http://developer.android.com/sdk/compatibility-library.html#SettingUp

2. Add the 'httpclient-4.1.1.jar' to your project build path located in Core/libs folder.

3. Add the 'httpcore-4.1.jar' to your project build path located in Core/libs folder.

4. Add the 'httpmime-4.1.1.jar' to your project build path located in Core/libs folder.
